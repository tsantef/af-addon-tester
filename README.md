af-addon-tester
===============

<img src="http://appfog.com/images/logo.png" />

Allows developers to test App Fog add-ons

## Setup ##

Install it like any Ruby Gem:

    $ gem install af-addon-tester


## Usage ##

    $ af-addon-tester
